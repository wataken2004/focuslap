# FocusLap 使い方ガイド 🐠

**集中すると魚が増えるタスク管理アプリ**
https://focuslap-iggi.vercel.app

集中した時間に応じて魚が集まり、図鑑が埋まっていきます。途中でスマホを触ると魚が逃げるので、サボり防止になります。無料・インストール不要です。

---

## 1. まずはホーム画面に追加（1分で完了）

ブラウザのまま使うより、ホーム画面に追加すると**通知が届く・全画面で使える・アプリと同じ感覚**になります。

### iPhoneの場合
1. **Safari**で https://focuslap-iggi.vercel.app を開く（Chromeではなく必ずSafariで）
2. 画面下の**共有ボタン**（□から↑が出ているアイコン）をタップ
3. メニューを下にスクロールして「**ホーム画面に追加**」をタップ
4. 右上の「追加」→ ホーム画面に魚のアイコンが追加される
5. **以降はそのアイコンから起動する**（通知機能はホーム画面版でしか使えません）

### Androidの場合
1. **Chrome**で https://focuslap-iggi.vercel.app を開く
2. 右上の「**︙**」メニューをタップ
3. 「**ホーム画面に追加**」（または「アプリをインストール」）をタップ
4. 「追加」→ 完了

## 2. ログイン

| 方法 | こんな人向け |
|---|---|
| **Googleでログイン** | 一番かんたん。1タップで完了 |
| **メールアドレスで登録** | Googleアカウントを使いたくない人 |
| **ゲストとして使う** | お試し用。データはその端末のブラウザにしか残らない |

Googleかメールでログインするとデータがクラウドに保存され、PC・スマホなど**複数の端末で同期**されます。

## 3. 基本の使い方（最初の5分）

1. 下のメニューから「**集中**」タブを開く
2. 画面下の**魚カードから集中時間を選ぶ**（最初は5分🫧か10分がおすすめ）
3. 「**スタート**」を押して勉強・作業に集中
4. 時間になると🎉お祝い画面とともに**魚を獲得！**
5. 「**水槽**」タブで獲得した魚が泳いでいるのを眺める

### 大事なルール

- ⚠️ **集中中に1分以上アプリを離れると魚が逃げます**（画面スリープはOK）
- 📱 スマホの単語帳アプリ等で勉強したいときは「**スマホ学習モード**」をONに → 他のアプリを開いても逃げなくなります
- 🔁 「**繰り返しモード**」をONにすると、休憩後に自動で次の集中が始まります

### 長時間やるなら「⏲ ストップウォッチ」

時間を決めずに計りたいときは、集中タブ上部で「⏲ ストップウォッチ」に切り替え。時間がカウントアップされ、「終了して記録」を押した時点の時間で魚がもらえます。**長くやるほどレアな魚**（1時間でサメ🦈、2時間で恐竜🦕…）。

## 4. 各タブでできること

| タブ | できること |
|---|---|
| **集中** | タイマー／ストップウォッチ、今日の集中量、週間グラフ、勉強の後から記録 |
| **タスク** | やることリストの管理。期限・開始時刻・目標との紐付け |
| **予定** | カレンダーで日付ごとにタスクを配置 |
| **目標** | 長期目標・仕事プロジェクトを作って、合計集中時間と獲得魚を確認 |
| **水槽** | 獲得した魚が泳ぐ水槽、魚図鑑、完了タスクの振り返りメモ |

各タブのタイトル横の「**?**」を押すと、そのタブの説明がいつでも見られます。

---

## 5. 通知を受け取る（アプリを閉じていてもOK）

1. **ホーム画面に追加したアプリ**から開く（重要）
2. 右上の「**⚙️**」→「📲 プッシュ通知」を「**有効にする**」→ 許可

これで以下が届きます：
- ⏱ **集中終了**（タイマー満了をお知らせ）
- ⏰ **タスクの開始時刻**（設定した時刻の前にお知らせ）
- 📝 **未完了リマインド**（🔔をONにすると1時間ごと・朝8時〜夜10時）

※通知は最大5分ほど遅れることがあります。マナーモード中は音は鳴りません。

## 6. 活用例

### 🎓 大学生：授業・課題

```
目標：「経済学B 単位取得」（🎯目標・期限 7/31）
  ├ タスク：第5回レポート（期限 6/20・開始 19:00）
  ├ タスク：教科書 ch.6 読む
  └ タスク：過去問1年分

→ 授業後に「レポート」タスクを選んで25分🐡×4セット
→ 目標を開くと「合計6時間・🐡×8 🦑×2」のように積み上げが見える
```

### 💼 就活

```
目標：「夏インターン選考」（🎯目標・期限 6/30）
  ├ タスク：ES下書き（A社）（期限 6/18・開始 20:00）← 20時に通知が来る
  ├ タスク：Webテスト対策 非言語
  ├ タスク：企業研究ノート（B社）
  └ タスク：面接想定問答の作成

→ ES執筆は時間が読めないので⏲ストップウォッチで計測
→ 終わったタスクにメモ：「ガクチカは結論を先に書く方が読みやすかった」
→ 水槽タブの振り返りが、そのまま就活の記録ノートになる
```

### 🏢 新卒・社会人：仕事と資格

```
目標①：「簿記2級 合格」（🎯目標・期限 11月試験日）
  └ 平日夜に25分×2。スキマは📱スマホ学習モードON でアプリ学習

目標②：「営業部の業務」（💼仕事）
  ├ タスク：月次資料の作成（期限・開始 9:30）
  ├ タスク：商品マニュアル読み込み
  └ タスク：日報テンプレ改善

→ 仕事は💼、自己研鑽は🎯と分けると、「今月仕事に何時間、勉強に何時間使ったか」が一目でわかる
→ 完了メモに「この資料は前月コピーで時短できた」と残せば翌月の自分が助かる
```

## 7. よくある質問

**Q. 魚が逃げた！**
集中中に1分以上アプリを離れたためです。スマホで勉強するときはスマホ学習モードをONに。なお逃げた数は毎日リセットされます。

**Q. 図鑑の「？？？」は何？**
まだ獲得していない魚です。必要な集中時間のヒントだけ見えます。獲って確かめてください。

**Q. 完了音が鳴らない**
スマホがマナーモードになっていないか確認してください。

**Q. データが消えたりしない？**
Google/メールでログインしていればクラウドに保存され、端末を変えても引き継がれます。完了したタスクとメモは、タスクを削除しても水槽の振り返りに永久に残ります。

**Q. お金はかかる？**
完全無料です。
