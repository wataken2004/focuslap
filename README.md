# FocusLap

集中すると魚が育つ勉強管理PWA。

## セットアップ

### 1. 依存パッケージをインストール

```bash
npm install
```

### 2. Firebase の設定（任意・なくてもゲスト利用可）

1. [Firebase Console](https://console.firebase.google.com/) でプロジェクトを作成
2. Authentication → ログイン方法 → Google を有効化
3. Firestore Database を作成（テストモードでOK）
4. プロジェクト設定 → アプリ → ウェブアプリを追加 → 構成をコピー
5. `.env.example` を `.env.local` にコピーして値を入力

```bash
cp .env.example .env.local
# .env.local を編集してFirebase設定を入力
```

### 3. 開発サーバー起動

```bash
npm run dev
```

### 4. ビルド（GitHub Pages / Netlify 等にデプロイ）

```bash
npm run build
# dist/ フォルダをデプロイ
```

---

## 機能

- **ポモドーロタイマー** — 集中 / 休憩を繰り返す
- **魚ずかん** — セッション時間が長いほど珍しい魚が獲れる（5分〜120分で11種類）
- **アプリ切り替え検知** — タイマー中にアプリを離れると自動停止
- **Googleログイン** — 複数端末でデータを同期
- **Googleカレンダー連携** — カレンダータブでイベントを表示・追加
- **ゲストモード** — ログイン不要でブラウザ内に保存
