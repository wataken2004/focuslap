# FocusLap 🐠

集中すると魚が増えるタスク管理PWA。

**本番URL**: https://focuslap-iggi.vercel.app
（スマホは「ホーム画面に追加」でアプリとして使えます）

## 主な機能

- **⏱ ポモドーロタイマー** — 魚カードで集中時間を選択。完了で魚を獲得、紙吹雪のお祝い演出＆ファンファーレ
- **⏲ ストップウォッチ** — カウントアップで勉強時間をそのまま記録。終了時点の時間で魚を獲得
- **🐟 魚ずかん（11種）** — 5分🫧〜2時間🦕。集中が長いほどレアな魚。未獲得はシルエットでネタバレ防止
- **🎯 目標・仕事管理** — タスクを紐付けて進捗・集中時間・獲得魚を目標ごとに集計
- **📅 カレンダー** — 月/週/日表示、日付へのタスク割り振り
- **🔍 振り返り** — 完了タスクは永久アーカイブ。タップでメモを記録（タスクを消しても残る）
- **📲 プッシュ通知** — アプリを閉じていても「集中終了」「開始時刻」「未完了リマインド」が届く
- **👤 アカウント** — Google / メール+パスワード ログインで複数端末同期。ゲストモードも可
- **🎮 ゲーム性** — 1分以上アプリを離れると魚が逃げる（📱スマホ学習モードで免除）、🔁繰り返しモード

## 技術構成

| 役割 | サービス |
|---|---|
| フロントエンド | React + Vite（PWA） |
| ホスティング | Vercel（mainへのpushで自動デプロイ） |
| 認証・DB | Firebase Auth / Firestore |
| プッシュ通知 | Web Push（VAPID）＋ GitHub Actions ＋ cron-job.org |

詳細な運用・トラブル対応は [OPERATIONS.md](./OPERATIONS.md)、開発者向け仕様は [CLAUDE.md](./CLAUDE.md) を参照。

## セットアップ（開発）

```bash
npm install
cp .env.example .env.local   # Firebase設定とVAPID公開鍵を入力
npm run dev                  # http://localhost:5173
```

`.env.local` 未設定でもゲストモード（localStorage保存）で動作します。

```bash
npm run build                # 本番ビルド（dist/）
node scripts/make-icons.mjs  # アイコン変更時にPNG再生成
```
